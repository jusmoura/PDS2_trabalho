\doxysubsection{Detailed Description}
Controlador responsável por gerenciar operações relacionadas aos jogadores. 

A classe {\ttfamily \mbox{\hyperlink{classPlayerController}{Player\+Controller}}} lida com a adição, remoção, e acesso de jogadores dentro do sistema. Ela também fornece métodos para listar jogadores, tanto por nome quanto por apelido (nickname).
