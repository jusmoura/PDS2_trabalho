\doxysubsection{Detailed Description}
Gerencia a interação com o usuário através de menus e operações relacionadas a jogadores e jogos. 

A classe {\ttfamily \mbox{\hyperlink{classInterface}{Interface}}} fornece métodos para exibir menus, criar e remover jogadores, realizar login de jogadores, e gerenciar a interação geral com o usuário. Ela se comunica com outras classes, como {\ttfamily \mbox{\hyperlink{classPlayerController}{Player\+Controller}}}, para manipular dados dos jogadores e iniciar diferentes jogos.
