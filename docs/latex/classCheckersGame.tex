\doxysubsection{Detailed Description}
Implementa a lógica do jogo de damas (checkers) baseado na classe \mbox{\hyperlink{classBoard}{Board}}. 

A classe {\ttfamily \mbox{\hyperlink{classCheckersGame}{Checkers\+Game}}} é uma implementação específica de um jogo de damas que herda da classe {\ttfamily \mbox{\hyperlink{classBoard}{Board}}}. Ela inclui métodos para definir o tabuleiro, validar movimentos, realizar jogadas e determinar o vencedor do jogo.
