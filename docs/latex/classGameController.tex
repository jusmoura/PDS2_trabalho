\doxysubsection{Detailed Description}
Gerencia a lista de jogos disponíveis no projeto. 

A classe {\ttfamily \mbox{\hyperlink{classGameController}{Game\+Controller}}} é responsável por manter uma lista de jogos disponíveis e fornecer acesso a esses jogos. A lista de jogos é gerenciada internamente e pode ser expandida com novos jogos adicionados à lista.
