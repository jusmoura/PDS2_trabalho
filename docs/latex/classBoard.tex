\doxysubsection{Detailed Description}
Classe abstrata que define a estrutura e funcionalidades básicas de um tabuleiro de jogo. 

A classe {\ttfamily \mbox{\hyperlink{classBoard}{Board}}} é uma classe abstrata que define as operações essenciais para manipular um tabuleiro de jogo. As classes derivadas devem implementar o método virtual puro {\ttfamily play} para realizar jogadas específicas do jogo.
