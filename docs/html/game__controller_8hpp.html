Detailed Description

Declaração da classe GameController para gerenciar jogos disponíveis no projeto e possibilitar o salvamento de informações estatísticas dos players.

Este arquivo define a classe GameController, que gerencia a lista de jogos disponíveis no projeto e possibilita o salvamento de informações estatísticas dos players. A classe mantém uma lista de jogos cadastrados e fornece métodos para acessar as estatísticas desses jogos.
