Detailed Description

Gerencia a leitura e escrita de dados de jogadores em arquivos.

A classe DatabaseController é responsável por carregar e salvar informações sobre jogadores a partir e para arquivos. Ela utiliza os métodos de leitura e escrita para interagir com a classe PlayerController.
