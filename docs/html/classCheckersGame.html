Detailed Description

Implementa a lógica do jogo de damas (checkers) baseado na classe Board.

A classe CheckersGame é uma implementação específica do jogo de damas que herda da classe Board. Ela inclui métodos para definir o tabuleiro, validar movimentos, realizar jogadas e determinar o vencedor do jogo.
