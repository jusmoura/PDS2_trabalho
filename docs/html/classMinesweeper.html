Detailed Description

Implementa o jogo Campo Minado.

A classe Minesweeper herda da classe Board e fornece a lógica específica para o jogo Campo Minado, incluindo a colocação de bombas, verificação de derrota e vitória, e exibição do tabuleiro com informações sobre as células adjacentes.
